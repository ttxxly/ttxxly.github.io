Java Collection 接口概述

2019-02-28

Collection 接口是层次结构中的根接口。构成 Collection 的单位称为元素。 Collection 接口通常不能直接使用，但该接口提供了添加元素、删除元素、管理数据的方法。由于 List 接口与 Set 接口都继承了 Collection 接口，因此这个方法对 List 集合和 Set 集合都是通用的。
